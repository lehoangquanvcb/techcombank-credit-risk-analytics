# Techcombank Interview Demo Talking Points

Author: Le Hoang Quan

Suggested pitch:
I built this end-to-end Credit Risk Analytics Platform using Excel, Python, SQL, Power BI design and Streamlit. It demonstrates portfolio risk monitoring, IFRS9 ECL, PD/LGD/EAD analytics, stress testing, early warning and executive dashboarding.

Demo flow:
1. Open the Streamlit app.
2. Point out the author label.
3. Ask the interviewer to click each numbered tab.
4. Switch between PC / Boardroom and Mobile friendly mode.
5. Run Base, Mild Stress and Severe Stress scenarios.
6. Explain management actions based on ECL and Stage migration.
