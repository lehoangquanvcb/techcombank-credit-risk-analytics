
# Interview Demo Script

## 30-second pitch
I built an integrated Credit Risk Analytics and IFRS9 modelling platform combining SQL, Python, Excel, Power BI design and Streamlit to simulate how a commercial bank can manage portfolio credit risk, calculate PD/LGD/EAD, estimate IFRS9 ECL, run macro stress testing, and translate model outputs into CRO-level recommendations.

## Demo flow
1. Open Excel dashboard and show total EAD, ECL, weighted PD/LGD, Stage 2 and Stage 3 ratio.
2. Show macro scenario assumptions and explain editable inputs.
3. Show loan portfolio data and IFRS9 staging logic.
4. Show stress testing: Base vs Mild vs Severe and ECL increase.
5. Show early warning list and management actions.
6. Explain Python model engine: logistic regression, XGBoost challenger, AUC/Gini/KS/PSI.
7. Explain SQL data mart and how it would feed Power BI in production.

## Key interview messages
- I understand both the modelling side and banking decision-making side.
- I can translate macro stress into portfolio risk and IFRS9 ECL.
- I can bridge Excel, Python, SQL, Power BI and executive reporting.
- The model is not meant to replace bank systems; it is a prototype showing architecture, logic and risk analytics thinking.
