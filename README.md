# Techcombank Credit Risk Analytics Platform Full v3.1

**Author:** Le Hoang Quan

This package is an interview-grade credit risk analytics and modelling platform for the Senior Expert, Credit Risk Analytics and Modelling role.

## New in v3.1
- Added author label: Le Hoang Quan
- Added user guidance telling users to click each numbered tab
- Added Streamlit Mobile friendly viewing mode
- Added deployment notes for GitHub and Streamlit Cloud

## How to run Streamlit
```bash
pip install -r requirements.txt
streamlit run app.py
```

## GitHub / Streamlit Cloud
- Push all files to GitHub
- On Streamlit Cloud, set main file path to `app.py`

## User Guidance
After opening the app, please click each numbered tab:
1. Executive Overview
2. Portfolio Quality
3. IFRS9 ECL
4. Stress Testing
5. Early Warning
6. Model Validation
7. Management Actions

Use the sidebar to switch between:
- PC / Boardroom
- Mobile friendly
